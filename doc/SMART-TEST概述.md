## 项目概述

SMART-TEST 是一个 AI 辅助的智能软件测试平台，定位为：

- **人机协作测试平台**：通过 Web 界面和 MCP 协议，实现人类测试人员与 AI 助手的无缝协作
- **测试全生命周期管理**：覆盖需求分析、测试设计、用例管理、执行跟踪、缺陷管理全流程
- **知识中心**：构建统一的测试知识库，支持多版本、多语言管理

## 核心能力

### 1. AI 能力集成
- 通过 MCP 协议灵活对接第三方 AI 助手（如 GitHub Copilot）
- 接入全球领先供应商的旗舰 LLM 模型，获得最强 AI 能力
- Web 端实现 AI 与人的工作流程统一

### 2. 项目知识中心
- **多源文档整合**：支持多源文档导入，通过 AI 整理和人工审核，形成"单一事实来源"（Single Source of Truth）
- **AI 辅助设计**：测试观点、测试用例、用例执行、结果分析等环节均支持 AI 设计+人工评审的迭代模式，并实现知识回流
- **用例库统一管理**：支持多版本、多语言的用例库管理，利用 AI 进行数据对齐、语言翻译、冲突消解

### 3. 测试执行与管理
- 基于用例库筛选创建测试执行计划
- 统一跟踪手动测试和自动化测试
- 支持基于 Playwright 的 Web 自动化测试执行
- AI 辅助缺陷管理和质量分析

## 技术架构特点
- 前后端分离架构（Go 后端 + React 前端）
- MCP 服务器支持 AI 工具集成
- 灵活的版本管理和多语言支持
